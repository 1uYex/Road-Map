# Maintenance Protocol

## Daily: 3 Minutes

Create one daily log from [[Daily Progress Log Template]].

Record:

- main focus
- one evidence line
- next small unlock

Do not write a diary unless you want to.

## Weekly: 20 Minutes

Ask:

1. Which main quest moved?
2. Which active side quest supported it?
3. Which side quest should sleep?
4. Did any tier genuinely increase?
5. What remains locked?

## Monthly: 45 Minutes

Open [[RPG World Map.canvas]].

Change only:

- node status
- active side quest count
- unlock conditions
- one newly revealed node if earned

## System Collapse Prevention

If the vault feels too complex:

1. Stop adding new folders.
2. Freeze all side quests except one.
3. Use only the Canvas and daily log for one week.
4. Do not redesign the system during stress.

## Anti-To-do Hell Rules

- Never convert every desire into a task.
- A quest is a direction plus evidence, not a checklist.
- Daily tasks are disposable.
- Mastery tiers are slow.
- Main Quest beats novelty.
