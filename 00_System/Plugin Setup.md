# Plugin Setup

## Already Useful

### Canvas

Use for the world map, skill tree, and unlock geography.

### Dataview

Use for dashboards, active quest lists, locked regions, and recent evidence.

### Excalidraw

Use only for occasional conceptual maps, not daily management.

Good uses:

- physics concept maps
- research taste diagrams
- mathematical structure sketches

Avoid:

- turning every topic into a drawing
- maintaining huge beautiful maps that become a burden

## Recommended Optional Plugins

### Tasks

Use sparingly for weekly commitments only.

Rule:

- no giant task database
- no more than 3 active weekly tasks

### Metadata Menu

Use when editing YAML fields becomes annoying.

Useful fields:

- status
- tier
- prerequisite
- unlock_condition
- support_role

### Tracker

Use only for slow signals:

- running
- sleep
- bodyweight
- study days

Avoid tracking too many metrics.

## Minimal Plugin Philosophy

The system should work even if only Canvas and Dataview are active.
