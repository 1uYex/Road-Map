# Canvas Layout Guide

## Recommended Layout

Center:

- Main Quest: Become a Computational / Theoretical Physicist

Left wing:

- Mathematics as the abstract foundation
- Place higher mathematics farther outward

Right wing:

- Physics as the world model
- Place modern and research physics farther outward

Bottom:

- English as communication infrastructure

Outer ring:

- Computational Skills
- AI and Tools
- Quant and Finance
- Fitness
- Philosophy and Thinking

Far horizon:

- LOCKED / Fog / Unknown regions

## Node Colors

Use low-noise colors:

| Node Type | Color |
| --- | --- |
| Main Quest | near-white / graphite |
| Mathematics | cold blue-gray |
| Physics | pale violet-gray |
| English | muted green-gray |
| Side Quest | dim amber-gray |
| Hidden Quest | dark gray |
| Locked Fog | black / transparent gray |

## Connection Rules

- Thick lines: prerequisite links
- Thin lines: supportive links
- Dashed or dim links: future unlocks
- Do not connect everything to everything

## Hidden Region Representation

Use three layers:

- LOCKED: visible name is ????
- Fog: vague region name, no details
- Unknown: no node details, only unlock condition

This preserves the feeling of exploration without letting future fantasies steal attention.
