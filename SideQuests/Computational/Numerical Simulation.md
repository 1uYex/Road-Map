# Numerical Simulation

## Why This Supports the Main Quest

Simulation turns equations into observable behavior and trains physical intuition.

## Unlock Condition

Numerical Methods tier 2 + Python tier 3.

## Mastery Tiers

- 0: Can run a demo
- 1: Can simulate simple ODEs
- 2: Can visualize physical systems
- 3: Can test stability and error
- 4: Can design small computational experiments
- 5: Can support research-like projects
